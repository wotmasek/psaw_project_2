## Design System

### Theme Overview
The current theme is set to `light` mode.

### Color Palette
- **Primary Color:** `#2E7D32` (a vibrant green) is used for key interactive elements and calls to action.
- **Secondary Color:** `#E3F2FD` (a light blue) provides support for less prominent UI components.
- **Tertiary Color:** `#E8F5E9` (a very light green) serves as an additional accent, possibly for highlights or badges.
- **Neutral Color:** `#37474F` (a dark slate gray) forms the base for backgrounds and structural elements.

### Typography
- **Headlines:** `manrope` font family is chosen for headlines, providing a modern and clean aesthetic.
- **Body Text:** `inter` font family is used for all body text, ensuring readability and consistency.
- **Labels:** `inter` font family is also designated for labels, matching the body text for a unified look.

### Shape and Form
- **Roundedness:** UI elements feature a `moderate` roundedness (value `2`), offering a balanced feel between sharpness and softness.

### Spacing
- **Density:** The layout incorporates `normal` spacing (value `2`), providing a comfortable and clear arrangement of elements without being overly dense or sparse.
